Symple Unzipper Challenge

Have you ever received a pesky ZIP file and not been able to extract it? No worries! This web server can do it for you.

This web server has a REST API endpoint that will receive a ZIP file via HTTP POST and return a JSON object containing the decompressed contents of every file inside the ZIP.

The API documentation is available here. The documentation also includes forms to upload ZIP files directly from your browser.

The Challenge

Unfortunately, this server has a bug that can be exploited to exfiltrate local files. A flag has been saved in a file named flag.txt in the same directory as the server code. Your job is to exploit the bug and get the flag.

Solving the challenge does not involve denial of service or things like ZIP bombs. Please don't do that. HINT: There is a special type of file you will need to craft.

The Code

You can download the source code to the server (minus the flag file!). It contains a README.md with further instructions.
